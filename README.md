# 🍕 SQL Pizza Sales Analysis

## 🔥 Transformez des données brutes en insights stratégiques

Ce projet propose une analyse complète des ventes d'une pizzeria en 2022, en utilisant **SQL (DuckDB + JupySQL)** pour extraire des indicateurs clés de performance et guider la prise de décision.

---

## 📊 Les chiffres qui parlent

- **Analyse réelle sur 48 620 commandes**  
- **Chiffre d'affaires total :** `817 860 $`  
- **Identification des meilleures ventes par catégorie, taille et période**  
- **Mise en évidence des produits sous-performants**  
- **Requêtes SQL optimisées avec fenêtrage (`DENSE_RANK`, CTE, agrégations avancées)**
